Introduction

C is a general-purpose, procedural programming language that has been widely used since its creation in the early 1970s. It provides low-level access to memory and is known for its efficiency and portability. C is used to develop system software, embedded systems, compilers, and applications that require high performance.

C has a simple syntax but offers powerful features such as pointers, memory management, and direct interaction with hardware. It serves as the foundation for many modern programming languages, including C++, Java, and Python.

Key Features:

  • Procedural Programming: Uses structured programming concepts like functions and loops.
  • Efficiency: Compiled programs run faster and use fewer resources./li>
  • Portability: Code written in C can be compiled and executed on different platforms with minimal changes.
  • Low-Level Access: Provides direct access to memory via pointers.
  • Modularity: Supports functions and libraries for reusable code.
What You Should Already Know

Before learning C, it is helpful to have the following background knowledge:

  • Basic understanding of computers and programming concepts: Knowing how software interacts with hardware is beneficial.
  • Familiarity with mathematical logic and problem-solving:C involves working with numbers, loops, and logical conditions.
  • Experience with algorithms and flowcharts:Helps in understanding how C programs are structured and executed.
C and Other Languages

C has influenced many modern programming languages. Unlike Java, which is an object-oriented language with automatic memory management, C requires manual memory handling and follows a procedural approach. Understanding C helps in learning languages like C++, Java, and Python.

Hello World

To get started with writing C programs, create a simple "Hello, World!" program:

     #include <stdio.h>

            int main() {
                printf("Hello, World!\n");
                return 0;
            }
Variables

Variables store data in memory. In C, you must declare a variable before using it.


            int age = 25;
            float height = 5.9;
            char grade = 'A';
Declaring Variables

Variables are declared with a specific data type, such as int, float, char, etc.

Variable Scope

Scope defines where a variable can be accessed. C has local and global scopes.

Global Variables

Global variables are declared outside any function and can be accessed anywhere in the program.

Constants

Constants store fixed values that cannot be changed during execution.

                 #define PI 3.14159
                const int MAX = 100;
Data Types

C provides several data types:

  • int (Integer)
  • float (Floating point number)
  • char(Character)
  • double(Double-precision floating point)
if...else Statement

Conditional statements execute code based on conditions.


                if (age > 18) {
                    printf("You are an adult.\n");
                } else {
                    printf("You are a minor.\n");
                }
while Statement

Loops execute a block of code repeatedly while a condition is true.


                int i = 0;
                while (i < 5) {
                    printf("%d\n", i);
                    i++;
                }
Function Declarations

Functions modularize code for reusability and readability.


                int add(int a, int b) {
                    return a + b;
                }
